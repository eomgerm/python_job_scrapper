# Python Job Scrapper

Scrape the jobs from <a href="https://www.indeed.com">Indeed</a> and <a href="https://www.weworkremotely.com">We Work Remotely</a>.
You can check the list of jobs and download it by csv file.

## Tech Stack

### Back-end

1. Selenium
2. Beautiful Soup
3. Flask

### Front-end

1. Pico
2. Html
